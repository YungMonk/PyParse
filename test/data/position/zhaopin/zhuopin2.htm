职位描述

?工作职责：

1. 负责蚂蚁聚宝iOS／Android等主流移动设备操作系统的产品界面视觉设计；

2. 主导设定产品视觉风格；

3. 负责参与设计体验、流程的制定和规范；

4. 负责分享设计经验、推动提高团队的设计能力。

?任职资格：

1. 从事设计行业工作五年以上，具有大型软件产品界面风格设定以及VI设计经验；

2. 热爱设计，拥有宽广的行业视野与的审美标准，以及对设计前沿、设计趋势的高度敏感； 3. 具备良好合作态度及团队精神，并富有工作激情、创新欲望和责任感，能承受高强度的工作压力；

4. 对各种新鲜互联网产品和大众互联网产品有强烈兴趣并有灵敏触觉，富有创造力和激情。
