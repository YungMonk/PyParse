岗位职责

您实习期间的主要工作内容包括：
+ 深入了解和掌握客户公司、所在行业及空缺职位的背景情况
+ 结合客户提供的岗位描述，确定合理的搜寻范围
+ 通过不同渠道接触潜在候选人
+ 向候选人介绍相关的工作机会，并深入交流有关该职位的背景情况
+ 了解候选人的职业发展规划，并分享自己的专业意见
+ 对候选人的任职资格及任职意愿进行专业的评估
+ 将符合条件且有意愿的候选人推荐给客户公司

在实习中，您将有机会：
+ 了解跨国公司招聘高层管理人员的真实标准
+ 探索怎样规划成功的职场人生（每天接触真实的案例供您参考）
+ 学习怎样更加专业地书写和修改中、英文简历
+ 掌握与企业高层管理人员打交道的基本技巧，提高人际交往能力
+ 逐步掌握建立人脉网络和发展人际关系的技巧
+ 系统学习管理咨询、股权投资、消费品、零售、制造等行业的运营模式和管理知识
+ 练习规范的职场英文表达（书面为主）
+ 学会熟练使用Office办公软件
+ 获得优厚的待遇（底薪及佣金）

要求：
+ 人力资源、商务英语、文秘或相近专业
+ 本科（大四）或专科（大三）在读
+ 每周两个或以上的全天到办公室工作，其余时间可在校工作
+ 对猎头行业或人力资源管理岗位有兴趣和好奇心
+ 工作细心，有责任心
+ 性格外向、乐观、自信
+ 礼貌待人，谦虚好学，考虑他人感受，乐于助人，喜欢交朋友
+ 普通话标准，语音语调有较强的亲切感

工作地点：
+ 岚路567号品尊国际中心（地铁七号线岚皋路站）

有意者请发您的简历和照片至hr@shantonway.com。如果您符合我们的基本要求，我们会尽快与您联系。无简历或近照的应聘信我们无法接待，敬请谅解。
