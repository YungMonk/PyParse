职位描述

岗位职责：

  1. 根据项目需求进行PHP代码的设计与实现；

  2. 负责与产品设计人员沟通需求，参与方案设计；

  3. 负责单元测试；

  4. 负责解决运营中遇到的产品质量与性能问题；

  5. 参与技术文档的编写与维护。

任职条件：

  1. 精通PHP + MySQL编程，对PHP有较深的理解和应用，精通MVC开发模式及面向对象的开发思想；

  2. 熟悉至少一种主流的PHP框架（如：ThinkPHP、Yii、CodeIgniter、ZendFramework），并有使用该开源框架进行二次开发项目的成功案例；

  3. 精通HTTP协议，熟练掌握Web前端技术，熟悉掌握JavaScript、DIV+CSS布局、AJAX、JQuery等；

  4. 有开放平台的接入经验者优先（如：微信公众号开放平台，支付宝开放平台）；

  5. 拥有良好的代码习惯，结构清晰，命名规范，逻辑性强，代码冗余率低；

  6. 有良好的问题定位分析能力，对Web开发拥有浓厚的兴趣，能够积极主动跟进新技术。
