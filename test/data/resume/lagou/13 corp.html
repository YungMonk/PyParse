公司产品

数据科学产品

  • 网站

国双拥有一套完整的自主研发的数据科学产品系列，囊括网站/移动数据分析、广告数据分析及管理、用户体验追踪、归因引擎与数据整合、互联网舆情监测分析、新媒体数据分析、受众画像分析等七条产品线、十五个以上的数据科学产品，在数字广告、搜索营销、移动应用、电视媒体、运营商、电子政务等六大领域提供基于云计算和大数据大商业智能解决方案。

国双技术能力

  • 其他

国双擅长构建海量数据仓库，进行实时数据分析。十年来，国双积累了多项大数据技术，包括分布式处理、分布式文件系统、数据仓库及ETL、流数据处理、数据可视化等技术能力，逐步形成完整的技术架构，即：大数据平台、数据挖掘、机器学习、自然语言处理和数据可视化。

公司介绍

“  国双的核心业务覆盖商业企业、政府、新媒体三大领域。在商业市场，累计为包括微软丶HP丶可口可乐、欧莱雅、宝洁丶大众、希尔顿、中国国旅丶中国银联丶中国人保丶中国移动丶中国联通丶京东丶腾讯丶百度等遍布18个行业600多家国内外知名客户及品牌提供全流程数字营销技术和服务；在政府市场，2012年与国家信息中心联合成立网络政府研究中心，迄今已为全国3000多家政府机构提供大数据分析服务，这其中就包括“中华人民共和国中央人民政府门户网站”(www.GOV.cn)在内；在新媒体领域，与江苏网络电视台、湖南卫视、浙江卫视等20多家省级电视台和网络电视台建立合作，为其提供全媒体大数据分析，热播的《爸爸去哪儿》、《对话》、《中国好声音》中都有国双的技术支持。”

管理团队

  • 李峰

    国双联席总裁

    李峰先生于2009年8月加入国双，现任联席总裁，全面负责公司产品和服务的销售工作。作为中国资深的数字营销专家，李峰先生曾于2004-2008年间服务于百度公司，创建了百度大客户部，先后担任百度华东分公司总经理、百度大客户部总经理。他拥有中欧国际工商学院EMBA学位、对外经济贸易大学工商管理硕士学位，是北京邮电大学特聘教授。


  • 刘激扬

    国双首席技术官

    刘激扬先生2014年12月加入国双，任首席技术官（CTO）。加入国双前，刘激扬先生服务于微软公司长达16年，是微软亚洲互联网工程院（ Microsoft Search Technology Center Asia）的创建人，他创建的研发机构已从仅有数名工程师发展至近千人团队；他带领的软件性能优化团队，使微软Windows, Office, SQL Server 等核心产品线的性能获得了大幅提升。此前，刘激扬先生还曾在惠普公司总部担任软件设计工程师。

    刘激扬先生本科毕业于清华大学计算机系，后取得美国匹兹堡大学计算机科学硕士学位。


  • 杨韬

    国双首席咨询师兼高级副总裁

    杨韬先生九年来带领国双咨询团队和数据科学家团队为Microsoft丶Coca-Cola丶L’ORÉAL丶PlCC等众多世界500强在内的600多家商业客户或品牌提供基于云计算和大数据的商业智能（BI）咨询服务。同时，他还是国家信息中心网络政府研究中心首席数据科学家、百度营销学院特聘专家、中国人民大学特聘教授、北航软件学院特聘教授、SEMPO国际搜索营销专家协会组织在中国的首位认证专家。
